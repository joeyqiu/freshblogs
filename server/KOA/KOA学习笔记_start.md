### 目录结构介绍

在现在的脚手架里，创建一个src目录用来放置源文件，具体代码都在改目录下

src

 |--index.js   （启动脚本）

 |--render.js （模版渲染配置）

 |-- routes.js（路由配置）

 |----

​	|--views/ （模版文件）

​	|--routes/ （路由的具体文件部分）
