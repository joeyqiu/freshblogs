## Chalk

Terminal string styling done right.

终端展示有颜色的文字。



文档链接：https://www.npmjs.com/package/chalk



### Usage

```
const chalk = require('chalk');
console.log(chalk.blue('Hello world!'));
```

在终端显示蓝色的文案。
