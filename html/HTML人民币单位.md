## HTML中的人民币单位

**网页上的人民币标识** **[￥ ](http://en.wikipedia.org/wiki/¥)****请统一使用转义字符 `(HTML: &#165; or &yen; )`。

**直接写中文字符**　 **¥** **或　￥**　  **无论你用全角还是半角都是不规范的写法！**



| 字符 | 十进制   | 转义字符 |
| ---- | -------- | -------- |
| ￥   | `&#165;` | `&yen;`  |



一般都是使用转义字符`&yen;`



空格的转义符号： `&nbsp;`
